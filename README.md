# ownsha2
minimum ELF64 program to calculate its own SHA256.

## Usage

compile
```shell
# compile
make # or make final
```

clear
```shell
# clear
make clean
```

## Feature
- only syscall
- minimalist file
- no share libary


final size: `1290`
